## Packages
leaflet | Core mapping library
react-leaflet | React bindings for Leaflet
@types/leaflet | TypeScript definitions for Leaflet
date-fns | Formatting for the time widget
lucide-react | Icons for the UI

## Notes
- react-leaflet requires leaflet.css to be loaded, which is done via CDN import in index.css
- The application uses a dark mode default theme
- WMS layer for Iowa Mesonet radar is used for the weather overlay
